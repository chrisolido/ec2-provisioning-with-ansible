AWS EC2 Provisioning by Ansible
=======================================

EC2 instances are created or terminated with the Ansible Plays in this
repository, therefore software stack deployment is achieved on Amazon Cloud
with dynamic computing resources. Ansible dynamic inventory and cloud modules
are used to manage cloud resources such as security groups, VPC (Virtual
Private Cloud) with public ip addresses.
